Example Pentesting Script: Simple Port Scanner (Python)

import socket

def scan_port(ip, port):
    sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    sock.settimeout(1)
    result = sock.connect_ex((ip, port))
    sock.close()
    return result == 0

ip = '192.168.1.1'
ports = [22, 80, 443]

for port in ports:
    if scan_port(ip, port):
        print(f"Port {port} is open on {ip}")
    else:
        print(f"Port {port} is closed on {ip}")

Writeup: GreyHack Level 6 Challenge

In this Level 6 challenge, I had to gain root access on a target deep within a segmented network protected by multiple routers and firewall rules. The mission required pivoting, deep enumeration, and chaining multiple vulnerabilities.

Key steps included:

  • Initial access through a compromised external machine and scanning the local LAN using nmap via pivoted access.
  • Enumerated a router using ifconfig and FileExplorer.exe to identify internal IPs, then exploited it using an authenticated vulnerability to gain root access.
  • From the router, discovered an isolated host not reachable directly from my machine. I launched a targeted attack from the router itself to bypass external firewall detection.
  • Disabled the firewall via command-line by modifying configuration files from the router using root access (e.g., nano /etc/firewall.rules or custom exploit module).
  • Used a multi-stage exploit chain: a known vulnerability in an exposed HTTP service led to command execution, which I escalated to root using a local privilege escalation exploit stored in my toolkit.

This challenge taught me how to chain multiple systems to achieve a goal (realistic lateral movement), the importance of post-exploitation cleanup, and the risks of insecure router configurations in segmented environments.
